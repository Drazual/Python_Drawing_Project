	User Stories

-As a user I should be able to change brush size.

-As a user I should have a GUI to work with.

-As a user I should have different colors to chosoe from.

-As a user I should able to save current work.

-As a user when I close the application I should be prompted to save if I havn't already and if there are any changes.

-As a user I should be able to alter the canvas size.

-As a user I should be able to change the background color.

-As a user I should be able to undo or undo the undo.

-As a user I should be able to change opacity of brush.
